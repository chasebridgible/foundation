Co-evolution rule

Required: when the descriptive spec changes, this test spec changes. When this test spec changes, code-level tests are regenerated or updated. Code, specs, and test specs must ship in the same commit for behavior changes.

During review, search the changed descriptive and technical sections listed above. If they changed without this file changing, the diff is incomplete unless the final evidence explains why no verification contract changed.
