Foundation Backfill Define Spec Jobs Technical Spec

This technical layer makes Foundation Backfill Define Spec Jobs deterministic enough for scripts and structured enough for agent judgment.

Status: active
Updated: 2026-05-29
Parent: foundation.backfill-spec-job-queue.job

Implementation must preserve the source-of-record split: JSONL is canonical for layer rows and eval receipts, while HTML summaries are generated for human reading. Each command must be resumable from artifacts and must not rely on chat memory.

Contracts

Checker and fill loop

  1. Read active report, run log, upstream artifacts, and current layer artifact.
  2. Run the upstream semantic audit: terminal capabilities must have identity anchors, and needs-split capabilities must have concrete splitCriteria anchors before queue work proceeds.
  3. Select next pending or failed batch by deterministic order plus risk priority.
  4. Inspect only the evidence needed for the batch and update canonical JSONL rows.
  5. Run the checker; repair structural failures before semantic eval.
  6. Run or append eval receipts; generate HTML summary from JSONL and store the queue artifact fingerprint in the eval receipt.
  7. Revise failed rows until the gate passes or a named human decision blocks progress.
