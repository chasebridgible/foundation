Long-Running Agent Work

Long-running agent work can stay useful across time, context limits, restarts, handoffs, and review because the work is bounded, stateful, checked, and evidenced.

Status: active
Updated: 2026-06-20
Canonical section: #capability-intent

Capability intent

Foundation should make agents capable of doing sustained useful work over long durations. The broad concept is simple: long work stays trustworthy when it has a goal, bounded units, durable state, appropriate clocks, checks, and a handoff that lets the next actor continue.
