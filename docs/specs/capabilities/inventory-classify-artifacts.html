Inventory and Classify Repository Artifacts

This capability prevents agents from guessing what files matter by requiring a source-backed inventory first.

Status: active
Updated: 2026-06-01
Canonical section: #capability-intent

Capability intent

Foundation must help agents read a repository before writing specs. Artifact inventory records what exists, what each artifact appears to do, how it should be classified, and whether it is evidence, implementation, config, generated output, or noise.

FieldContractEvidence or source
