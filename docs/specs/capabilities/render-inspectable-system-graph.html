Render the System as an Inspectable Graph

This capability turns the Foundation substrate into a map that can be inspected instead of only read.

Status: active
Updated: 2026-06-01
Canonical section: #capability-intent

Capability intent

Foundation must render business and software architecture from source specs, not from a separate diagram. The graph is a decision surface: what can the system do, who or what does the work, how work moves, what evidence proves it, and where it is fragile.
