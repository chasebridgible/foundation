Operating Lessons Improve The System

Evidence from shipped-system operation becomes revisions to specs, jobs, evals, context, skills, and validators instead of staying trapped in incidents or chat.

Status: draft
Updated: 2026-06-06
Canonical section: #capability-intent

Capability intent

Foundation should make operating lessons durable enough that real-world failures, surprises, customer outcomes, and monitoring signals improve the substrate instead of becoming one-off knowledge.
