Compound Improvements Safely

This capability makes Foundation improve with use instead of drifting as agents add one-off fixes.

Status: active
Updated: 2026-06-01
Canonical section: #capability-intent

Capability intent

Foundation must absorb useful process improvements while preserving reviewability. Changes go through branch, PR, checks, graph validation, semantic review, merge, and local sync. Accepted lessons update the durable substrate that future agents load.
