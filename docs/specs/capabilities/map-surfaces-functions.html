Map Surfaces and Functions

This capability turns classified files into a behavior map agents can use before naming capabilities or jobs.

Status: active
Updated: 2026-06-01
Canonical section: #capability-intent

Capability intent

Foundation must connect artifacts to visible surfaces, functions, actions, and operating behaviors. Surface/function mapping preserves what users or operators can see, where the behavior is implemented, and which source artifacts support that reading.
