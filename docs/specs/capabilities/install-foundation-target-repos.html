Foundation Is Installable In Target Repos

This capability lets Foundation be reused across repos while keeping project facts inside the target repo.

Status: active
Updated: 2026-06-02
Canonical section: #capability-intent

Capability intent

Foundation must provide a repeatable way to set up target repositories with a small local adapter, spec registry, graph-compatible docs, shared HTML navigation, pinned checks, and a doctor command that catches drift.

Interfaces and sources

  • skills/install-foundation-substrate/SKILL.md executes target repo setup.
  • scripts/foundation-doctor.mjs validates target repo installation.
  • docs/site-nav.js provides the shared HTML docs navigation and sidebar collapse behavior that target repos copy or pin locally.
  • Target AGENTS.md files remain small adapters.
