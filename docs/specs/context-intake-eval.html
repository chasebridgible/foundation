Context Intake Eval Spec

Accept context intake only when an Omi payload produces local source evidence, normalized queryable records, a reviewable note, and no committed raw private state.

Status: active
Parent: foundation.context-intake.job
Updated: 2026-06-15

IDConditionMaps toMust prove
