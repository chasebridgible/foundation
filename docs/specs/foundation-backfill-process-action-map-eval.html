Foundation Backfill Process / Action Map Eval Spec

This test layer defines the deterministic and semantic gates that prove Foundation Backfill Process / Action Map can hand off safely.

Status: active
Updated: 2026-06-03
Parent: foundation.backfill-process-action-map.job

The tests treat the layer as a reproducible unit: structure is checked mechanically, semantic quality is judged through row-level outstanding eval receipts, and the HTML summary is a derived review surface.

Eval contract

scores process/action rows against Context Packs to detect missing states, hidden permissions, unmodeled errors, architecture leakage, missing downstream spec targets, and under-described visible behavior. The canonical output is JSONL with one summary receipt and one current receipt per evaluated row. The HTML summary is generated from JSONL so humans can read counts, failures, outstanding row coverage, revision targets, and examples.

  • Every blocking finding or warning names the row ID, evidence source where available, failure category, and revision target.
  • A row is outstanding only when deterministic checks pass for the current row, the row score is 100, the row fingerprint matches the receipt, no blocking findings exist, no warnings exist, no revision targets remain, and the row can support Author Specs without rediscovering source for core behavior.
  • Process / Action Map rows may not advance by aggregate score; every non-pending row must have its own current outstanding row receipt.
  • Row eval cannot replace the checker step. The eval command must fail when the Process / Action Map check artifact is missing, stale, or failing for the current fingerprint.
  • Rows with severe uncertainty must remain non-terminal or carry a named human decision.
  • The Foundation readiness gate is command/test readiness against completed Context Packs; Sandia Oil remains a later calibration run for real-repo quality tuning.

Gaps and follow-ups

  • Implementation commands now exist in Foundation and should be invoked from target repos that have completed Context Pack handoff artifacts.
  • Sandia Oil starts as calibration evidence, not a perfect gold fixture or a prerequisite for this layer's readiness.
  • Schema or rubric changes must update job, technical, and eval specs together.
