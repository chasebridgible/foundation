Request Customer Review Job

Ask satisfied customers for a review without contacting unresolved issue cases.

Capability supported

Supports Protect customer relationship.

Current handling

The customer coordinator sends review requests after work is closed and no recovery case remains open.

ActorRole
Customer coordinatorOwns intake, scheduling, customer updates, and review requests.

Process

  1. Review closed jobs eligible for feedback.
  2. Suppress requests when unresolved issue or refund state exists.
  3. Send review request with correct channel.
  4. Record request state and customer response signal.

Context, tools, and interfaces

ToolUse in job
Review dashboardReview requests, customer satisfaction signals, and follow-up status.
Customer messagingConfirmations, arrival updates, issue recovery, and review requests.

Evidence and evaluation

A review request log tied to a closed job proves feedback was requested appropriately.
